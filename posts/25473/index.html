开发环境的搭建

JDK的下载

JDK即Java Development Kit,它对于系统能编译和执行Java程序是必备的.
首先打开Oracle公司的官网,可以看到如下界面

JDK的安装

一直点击下一步即可,但要记住JDK的安装位置,因为待会需要使用.
不推荐更改它的默认安装路径

JDK的配置

以Windows系统为例,JDK的配置步骤如下:

首先在此电脑/计算机的图标上点击右键,选择属性,在弹出的窗口左端点击高级系统设置超链接
在打开的系统属性对话框中选择高级选项,单击环境变量按钮.

然后单击新建,在弹出的对话框中变量名输入JAVA_HOME,变量值中输入你的JDK安装的完整路径,如下

Java编译器的下载安装

本文的Java编译器选择的是Eclipse,如有其它好用的编译器也可以.

Eclipse的下载

首先进入Eclipse的下载界面

找到Eclipse IDE for Java Developers,单击”64 bits”超链接.
注意别看错下载了Eclipse IDE for Java EE Developers
在新打开的界面点击”download”,等待下载即可.

Eclipse的安装

下载完成后,解压安装包即可使用eclipse.
